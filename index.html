Um pouco sobre mim

Sou Rodrigo Bragagnolo, estudante da FATEC ITU e apaixonado pela área da tecnologia. Em 2019, tive meu primeiro contato com a programação, mas foi em 2020, graças a um amigo, que me envolvi completamente com o desenvolvimento web. Desde então, tenho dedicado meu tempo ao estudo e aprimoramento nesse campo fascinante.

Sou encantado pela tecnologia por sua vastidão de possibilidades. Procuro sempre aprender mais, buscar soluções criativas e me dedicar a projetos empolgantes. Sinto que estou construindo uma base sólida para uma carreira promissora no desenvolvimento web.

Meus Projetos

Alimente Seu Monstro

Este projeto foi inspirado no jogo mobile chamado "Pou". O objetivo deste projeto é garantir que o monstro esteja sempre feliz e saudável. O projeto que desenvolvi permite que os jogadores experimentem a diversão e os desafios de cuidar de um monstro virtual.

Consumo de Energia

Este projeto tem como objetivo calcular de forma precisa e eficiente o consumo de energia dos aparelhos eletrônicos em uma casa. O site disponibilza a potência de cada aparelho e o usuário escolhe tempo de utilização, então o site fornece os resultados com clareza.

Finans

Apresento o projeto de simulação de companhia de finanças desenvolvido com dedicação e expertise. Com design elegante e funcionalidades abrangentes, nossa plataforma proporciona aos usuários a oportunidade de explorar diversos aspectos financeiros.

Vamos Contar

Este projeto é um site que interage com o usuário por meio de inputs. Permitindo a inserção de dois números onde, em seguida, o programa exibe todos os números entre esses dois valores. Essa funcionalidade é útil para visualizar uma sequência de números específica.
