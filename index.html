Um pouco sobre mim

Sou Rodrigo Bragagnolo, estudante da FATEC ITU e apaixonado pela área da tecnologia. Em 2019, tive meu primeiro contato com a programação, mas foi em 2020, graças a um amigo, que me envolvi completamente com o desenvolvimento web. Desde então, tenho dedicado meu tempo ao estudo e aprimoramento nesse campo fascinante.

Sou encantado pela tecnologia por sua vastidão de possibilidades. Procuro sempre aprender mais, buscar soluções criativas e me dedicar a projetos empolgantes. Sinto que estou construindo uma base sólida para uma carreira promissora no desenvolvimento web.

Meus Projetos

Alimente Seu Monstro

Este projeto foi inspirado no jogo mobile chamado "Pou". O objetivo deste projeto é garantir que o monstro esteja sempre feliz e saudável. O projeto que desenvolvi permite que os jogadores experimentem a diversão e os desafios de cuidar de um monstro virtual.

Rede Social Orbee

O Orbee é uma plataforma de rede social inspirada no Facebook, onde os usuários podem se cadastrar, criar perfis, conectar-se com amigos, compartilhar postagens e imagens, além de interagir por meio de comentários e curtidas.

Finans

Apresento o projeto de simulação de companhia de finanças desenvolvido com dedicação e expertise. Com design elegante e funcionalidades abrangentes, nossa plataforma proporciona aos usuários a oportunidade de explorar diversos aspectos financeiros.

Seguradora TopCar - CRUD

Este projeto é uma simulação de uma seguradora para carros, desenvolvido para fins educacionais e de demonstração. O usuário poderá criar um perfil, ler seus dados, editar as informações apagar sua conta (CRUD)

Clima & Tempo - API

Este projeto é um site que informa ao usuário informações sobre o clima do lugar que o indivíduo quiser basta apenas pesquisar o local e pronto! Este software utiliza 3 APIs que auxiliam o site a fornecer informações precisas, são elas: OpenWeather API, TimeZone DB e FlagsAPI

Gerenciador de Biblioteca - Java

Este projeto de Sistema de Gerenciamento de Biblioteca em Java permite a administração de livros, usuários e empréstimos. O objetivo principal é praticar conceitos básicos e intermediários de Java, como POO, manipulação de arquivos, estrutura de dados, etc. O sistema registra, atualiza e organiza os dados de forma simples, sem o uso de frameworks.
